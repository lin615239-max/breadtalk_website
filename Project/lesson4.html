Iteration - Loop, repetition

A loop runs the same code over and over again, as long as the condition is true.

For loop

A for loop is best to use when you know how many times the code should run.
To count, a for loop uses a counting variable, to keep track of how many times the loop has run.
Three parts of a for loop:

  • Initialization - Starting value
  • Condition - for loop runs as loog as the condition is true
  • Increment - Increase the counting variable

While loop

The while loop loops throught a block of code as long as a specified condition is true.

Do While loop

The do/while loop executes code once, then repeats while the condition is true,and always runs at least once, even if the condition is false.
